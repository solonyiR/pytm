# Terminal file manager pytm

Simple cross-platform (testing phase) CLI file manager
language: Python
main library: os

## Feachures:
- Browse folders
- Create / Remove folders
- Open files (testing phase)

## How to install
installing via pipx.

_For debian based:_
'''bash
sudo apt install python3-pipx
pipx ensurepath

# close terminal and open another one

pipx install git+https://github.com/solonyiR/pytm.git

pytm
'''

_For arch based:_
'''bash
sudo pacman -S python-pipx
pipx ensurepath

# close terminal and open another one

pipx install git+https://github.com/solonyiR/pytm.git

pytm
'''
